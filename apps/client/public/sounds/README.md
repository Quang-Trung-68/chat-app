# Âm thanh cuộc gọi

Thêm hai file (tùy chọn):

| File | Dùng cho |
|------|----------|
| `ringtone.mp3` | Nhạc chờ — **người gọi** khi đang chờ bên kia bắt máy (`outgoing`) |
| `belltone.mp3` | Chuông — **người nhận** khi có cuộc gọi đến (`incoming`) |

Đường dẫn public: `/sounds/ringtone.mp3`, `/sounds/belltone.mp3`.

Nếu thiếu file, trình duyệt sẽ không phát âm thanh (đã bắt lỗi khi play).
